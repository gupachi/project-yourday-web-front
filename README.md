# Yourday
온라인 축하 페이지 제작 & 공유 서비스

## 프로젝트 소개
몇 분 만에 완성도 높은 '하루 전용 축하 페이지'를 만들어, 단 하나의 링크로 지인들이 서로 공유하며 축하할 수 있는 웹 서비스입니다. 

## 프로젝트 상세 정보
- MVP빌더스클럽[mvp 개발 소모임] 
- 프로젝트 기간: 2025.12.23 ~ 2026.01.02
- 팀: 프론트[전지원] 1명, 백엔드 1명

## 주요 기능 
- 개인별 개별 축하 페이지 생성
- 링크 기반 공유
- 사진 업로드 및 방명록 
- 실시간 페이지 수정

## 역할 
- 웹 프론트엔드 개발 
- UI 구현 및 페이지 상태 관리
- Swagger 기반 API 연동

## 기술스택
